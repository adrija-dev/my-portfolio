# 📱 My Flutter Portfolio

A modern, responsive personal portfolio application built with Flutter to showcase my projects, skills, and experience.

## 🚀 Live Demo
👉 [View My Live Portfolio Website](https://adrija-dev.github.io/my-portfolio/)


## 🛠️ Built With
- **Framework:** Flutter (Dart)
- **Design:** Fully Responsive (Supports Desktop, Tablet, and Mobile layouts)
- **Deployment:** GitHub Pages

## 📂 Project Structure
- `lib/screens/largescreens/`: Layout designs tailored for desktop view.
- `lib/screens/mobiles/`: UI optimized for mobile platforms.
- `lib/screens/tablets/`: Layout specifications for tablet view.
- `lib/screens/widgets/`: Modular, reusable UI components (CV download, social section, skills widgets).

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/adrija-dev/my-portfolio.git`
2. Fetch dependencies: `flutter pub get`
3. Run the project: `flutter run`
